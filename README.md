# Pemberton & Hale — Real Estate Landing Page

A professional single-page landing page for a real estate agency. Built entirely with AI from a single voice prompt. No frameworks, no dependencies — just HTML, CSS, and JavaScript.

## Live Demo

[View Live Demo](https://provibecoding.app/resources/real-estate-template/demo/)

## What's Included

- `index.html` — 9 sections with semantic HTML and inline SVGs
- `style.css` — Full theming with CSS variables, mobile-first responsive design
- `script.js` — Dark/light mode toggle, smooth scroll, animations, mobile nav
- `assets/` — 16 optimized images

## Sections

1. Navigation (with mobile hamburger menu)
2. Hero
3. Testimonials (6 cards)
4. Services (6 cards)
5. About
6. How It Works (3 steps)
7. CTA
8. Contact Form
9. Footer

## Features

- Responsive (320px, 768px, 1024px breakpoints)
- Dark/light mode with localStorage persistence
- Scroll-triggered fade-in animations
- Hover effects on cards and buttons
- Google Fonts (Montserrat + Inter)
- Zero dependencies
